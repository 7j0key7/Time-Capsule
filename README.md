# Time-Capsule
Пак Владислав и Миллер Виктория
9-4-ПО-22
TimeCapsule - это мобильное приложение которое позволяет пользователю отправлять сообщения для самого себя в будущем. Основная цель данного приложения это зафиксировать различные мысли, цели или напоминание, какие-либо эмоции и т.п вещи, чтоб в дальнейшем ты мог получить данное сообщение в удобном для тебя способе и удобном для тебя времени. Ты можешь указать как способ получения этого, так и указать определённые даты.
